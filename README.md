New Frontend For theRARBG [On-Progress]

### Requirements
- Use Node 19.9.0
- Node.js 19+ and npm

### Getting started

Run the following command on your local environment:

```shell
npm install
```

Then, you can run locally in development mode with live reload:

```shell
npm run dev
```
